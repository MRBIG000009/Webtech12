Rappel #1

C'est quoi un site web ?

Un site web est un ensemble de pages connectées entre elles via des liens hypertextes, accessible via Internet. Il peut contenir du texte, des images, des vidéos et d'autres éléments interactifs.

Quel est son utilité ?

Les sites web sont utilisés pour fournir des informations, interagir avec des utilisateurs, vendre des produits, ou encore pour offrir des services en ligne.

Les méthodes pour créer un site web

Il existe deux grandes méthodes pour créer un site web :

  • Code : Cela consiste à utiliser un langage informatiquet pour construire un site.
  • No Code : Utiliser des plateformes comme Wix, WordPress ou Webflow qui permettent de créer des sites sans avoir à écrire de code.

Les méthodes pour créer un site web avec du code

Voici les langages de base requis pour créer un site web avec du code :

  • HTML : Le langage de balisage qui structure la page.
  • CSS : Permet de styliser les éléments HTML.
  • JavaScript : Rend la page interactive.

HTML

Le HTML (HyperText Markup Language) est utilisé pour structurer le contenu d'un site. Chaque élément est représenté par une balise.

Balises ouvrantes et fermantes

En HTML, la plupart des éléments sont entourés de balises ouvrantes et fermantes. Une balise ouvrante indique le début d'un élément, tandis qu'une balise fermante marque la fin de cet élément. Exemple :

<p>Ceci est un paragraphe</p>

Ici, <p> est la balise ouvrante et </p> est la balise fermante.

Balises HTML courantes

Voici quelques balises de base utilisées dans la création de sites web :

  • <h1> - <h6> : Balises de titre
  • <p> : Balise de paragraphe
  • <a> : Balise pour les liens
  • <button> : Balise pour les boutons
  • <form> : Balise pour les formulaires
  • <div> : Balise pour les sections de page
  • <nav> : Balise pour les barres de navigation

Relier deux pages HTML

Pour relier deux pages HTML, on utilise la balise <a> :

<a href="page2.html">Aller à la deuxième page</a>

Styliser une page HTML

On peut ajouter du CSS directement dans le HTML en utilisant la balise <style> :

<style>
body {
    background-color: #e9f7e9;
    color: #333;
}
</style>

Propriétés CSS courantes

Voici quelques propriétés CSS importantes pour styliser vos pages :

  • color : Modifier la couleur du texte
  • padding : Ajouter de l'espace intérieur à un élément
  • margin : Ajouter de l'espace extérieur à un élément
  • border : Ajouter une bordure autour d'un élément
  • border-radius : Arrondir les coins d'un élément
  • display : Modifier la façon dont un élément est affiché (block, inline, flex, etc.)
